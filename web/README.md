# Kozzy Owner PWA

Next.js 15 + Tailwind + Serwist PWA for PG owners. Connects to FastAPI backend at `/api/v2/app/*`.

## Dev

    npm install
    npm run dev

Open http://localhost:3000.

## Test

    npm run test      # Vitest (unit)
    npm run e2e       # Playwright (E2E, headed)

## Build

    npm run build && npm start
